# Linux Automation Tools 🐧⚙️

Welcome to the **Linux Automation Tools** repository, managed by **Ayutech IT Infrastructure**. This collection is designed to simplify system administration, monitor server health, and automate repetitive tasks for Linux environments.

## 🚀 Overview
As network and systems engineers, we understand the importance of uptime and performance. This repository contains enterprise-grade Bash scripts and automation workflows to ensure your servers are running optimally.

## 🛠️ Tools Included

### 1. Server Health Check (`health_check.sh`)
A lightweight script to get an instant snapshot of your server's vital signs.
* **Disk Usage:** Identifies partition levels to prevent storage-related crashes.
* **Memory Usage:** Checks RAM and Swap availability.
* **Uptime:** Monitors how long the system has been running.

## 📋 Prerequisites
- A Linux-based operating system (Ubuntu, CentOS, Debian, etc.)
- Basic terminal access with execution permissions.
